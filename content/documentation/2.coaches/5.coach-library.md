The **Library** is your content workspace as a coach. Build reusable workouts, exercise databases, and training plan templates — then assign them to athletes.

## Library sections

Access from **Coaching → Overview** quick actions or the sidebar **Library** menu:

| Section       | Purpose                                               |
| ------------- | ----------------------------------------------------- |
| **Workouts**  | Structured workout templates with intervals and zones |
| **Exercises** | Strength and mobility exercise database               |
| **Plans**     | Multi-week training plan templates                    |

## Creating library content

### Workout templates

1. Go to **Library → Workouts**
2. Click **Create Workout**
3. Define intervals, zones, duration, and sport type
4. Save to your library

### Plan templates

1. Go to **Library → Plans**
2. Create or import a plan structure
3. Define blocks, weeks, and session outlines
4. Save as a reusable template

## Sharing with athletes

Assign library content to connected athletes:

1. Open a plan or workout in the library
2. Click **Share** or **Assign**
3. Select the target athlete (or team)
4. The content appears in the athlete's plan or workout list

Coaches can also share plans via link without requiring a direct connection.

## Coach workflow tips

- **Build once, assign many** — create a base-building template and assign it to multiple athletes with individual FTP adjustments
- **Use Chat while acting as athlete** — refine assigned workouts in context of the athlete's recent data
- **Review compliance** — after assigning, monitor completion via the [Coaching Calendar](/documentation/coaches/coaching-calendar)

## Athlete library access

Athletes can browse **Library → Plans** to view plans shared with them or access public templates. Coach-created assignments appear in their [Training Plan](/documentation/athletes/training-plans) view.
